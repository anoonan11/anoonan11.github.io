Encryptor Android App

Keywords: Android Studio, Github, Java

In Fall 2022, I took CS 6300 at the Georgia Institute of Technology. While in this class, I created several different applications. One of which is an encryptor application that is hosted on Android. The application uses a simple substitution cipher to encrypt a given string. I created an application in which a user can add a string, adjust the encryption parameters and be outputted an encrypted version of this string. The application contains regular expression (i.e., regex checks) to ensure that valid strings are passed.

Unfortunately, the code for this class is not to be shared outside of Georgia Tech so I cannot link the source code.
