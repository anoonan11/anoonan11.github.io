Job Offer Comparison Android App

Keywords: Android Studio, SQLite, Github, Java

In Fall 2022, I took CS 6300 at the Georgia Institute of Technology. While in this class, I created several different applications. One of which is a job comparision application that is hosted on Android. With two other teammates, we created an application in which a user can add / edit their current job, add job offers, view a ranked list that is computed using comparison settings that the user sets, and select two jobs to compare line by line.

A feature we added was the ability to persist the user's information when they exit the application. We were able to enable this functionality by connecting the SQLite database within the Android device. Unfortunately, the code for this class is not to be shared outside of Georgia Tech so I cannot link the source code.
