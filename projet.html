Projet

Keywords: Angular, Docker, Javascript, web sockets, GraphQL, MongoDB

While at Capital One, I participated in the Capital One Developer Academy. As our Unit 4 project, we were required to create a fullstack project that used Angular as a frontend framework. On a team of five, we created Projet (not a typo!) which is a project managment site.

We used Angular for the frontend and created features like dynamic Gantt charts and a collaborative sticky note board that updated live for all users using web sockets. The backend used GraphQL and our database was MongoDB.

This application was developed at Capital One with their resources so the source code and demo is private.
