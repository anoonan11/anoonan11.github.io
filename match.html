Dinner Party


Keywords: HTML, CSS, Javascript, Docker

While at Capital One, I participated in the Capital One Developer Academy. As our Unit 1 project, we were required to create a game with the HTML, CSS, and Javascript that we learned at that point in the program. I created a traditional matching card game called Lost & Found. It came in 2 levels of difficulty and had a time limit.

This game was developed at Capital One with their resources so the source code and gameplay demo is private.
